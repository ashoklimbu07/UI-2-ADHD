# UI-2-ADHD

Interactive prototype — UI Version 2.

This is a fully interactive prototype (not just static mockups) of an ADHD-friendly planner app, generated from a Claude design. It includes complete, detailed features — calendar, to-do management, and insights — packaged as an installable PWA.

## Features

- Calendar and to-do integration
- Simple insights UI
- Installable as a Progressive Web App (manifest + service worker)
- Offline support via network-first with cache fallback

## Running locally

Serve the folder with any static file server, e.g.:

```bash
npx serve .
```

Then open the served URL in your browser.
